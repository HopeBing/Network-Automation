ACI Subnet Finder
-------------

Description
===========

  **ACI Subnet Finder locates and gets unicast gateway information**
  
Usage
======

**Enter Gateway:**

          >>> ACI Subnet Finder
              Total Fabric Subnets: 70
              Enter gateway IP plus prefix length (ex. 1.1.1.1)

          >>> Please enter a gateway IP: 10.10.10.1
              
          >>> BD:
                BD-VLAN1000
              Enable:
                yes
              Scope:
                public
              Unkwn Uni:
                flood
              Tenant:
                Customer-1
              APs:
                External
              Epgs:
                Device-Management
              L3Outs:
                L3OUT-To-Internet
              VRF:
                VRF-To-Internet
